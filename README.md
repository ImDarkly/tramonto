# Tramonto

Simple Mafia role assigner.

>[!CAUTION]
>Tramonto is currently in the early development stage!

## Features
- [ ] Random role assingment
- [ ] Customizable roles
- [ ] Role management
- [ ] Session tracking
- [ ] Theme switcher
- [ ] Hide killed player role


## Roles
- Civilian
- Doctor
- Detective
- Mafia
